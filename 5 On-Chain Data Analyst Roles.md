Here are the **5 common types of On-chain Data Analyst roles in Web3**. The title may be the same, but the **actual work and career path can be quite different.**

---

# 1️⃣ DeFi / Protocol Data Analyst

Works inside **DeFi protocols or blockchain ecosystems**.

### Main goal

Understand **how the protocol is used and how it grows.**

### Typical tasks

* Track **TVL (Total Value Locked)**
* Analyze liquidity pools
* Monitor user retention and growth
* Study yield farming activity
* Analyze governance participation

### Tools

* SQL
* Python
* Dune Analytics
* Flipside Crypto

Example companies

* Uniswap Labs
* Aave
* Lido

---

# 2️⃣ Crypto Market / Trading Analyst

Works for **trading firms, funds, or exchanges**.

### Main goal

Understand **capital flow and trading behavior.**

### Typical tasks

* Track whale transactions
* Analyze token supply movement
* Study market sentiment
* Identify trading opportunities

### Tools

* Python
* SQL
* Nansen
* Glassnode

Example companies

* Binance
* Kraken
* crypto hedge funds

---

# 3️⃣ Blockchain Intelligence / Forensics Analyst

Works in **security, investigation, and compliance.**

### Main goal

Track **illegal activity and scams on-chain.**

### Typical tasks

* Wallet clustering
* Track stolen funds
* Investigate hacks
* Identify laundering patterns

### Tools

* blockchain explorers
* graph analysis
* Python

Example companies

* Chainalysis
* TRM Labs
* Elliptic

---

# 4️⃣ Ecosystem / Growth Data Analyst

Works for **blockchain foundations or Layer-1 networks**.

### Main goal

Measure **ecosystem health and adoption.**

### Typical tasks

* Track new wallets
* Analyze developer activity
* Measure dApp usage
* Study network transactions

Example organizations

* Ethereum Foundation
* Solana Labs
* Polygon Labs

---

# 5️⃣ Product Data Analyst (Web3 Apps)

Works in **crypto apps, wallets, and Web3 platforms.**

### Main goal

Improve **product usage and user experience.**

### Typical tasks

* Analyze onboarding funnels
* Study user behavior
* Measure feature adoption
* Improve retention

Example companies

* Coinbase
* MetaMask
* OpenSea

---

# 💡 Which one pays the most?

Generally:

1️⃣ Crypto trading firms / hedge funds
2️⃣ Blockchain intelligence companies
3️⃣ DeFi protocol analysts
4️⃣ Product analysts
5️⃣ Ecosystem analysts

If you learn **SQL + Python + blockchain data**, you can move between these roles.
